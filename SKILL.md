# 交通事故案件AI办案工作流

## 概述

本Skill实现交通事故案件从证据到文书的端到端自动化：

1. **输入案件数据** → 原告/被告信息、医疗费、住院天数、伤残等级等
2. **自动计算赔偿** → 医疗费/护理费/误工费/死亡赔偿金等13项，含保险拆分
3. **智能推导证据** → 根据案件类型自动生成证据清单（27项规则库）
4. **三期评定参考** → 根据损伤描述自动匹配GA/T 1193-2014三期数据
5. **一键输出4份文件** → 赔偿明细报告.docx + 证据清单.docx + 要素式起诉状.docx + 赔偿明细.md

## 核心特性

- ✅ **样式100%还原**：基于原模板XML结构替换文本，绝非从零生成
- ✅ **官方数据校准**：河北省2023-2025年官方标准，含分行业工资19类
- ✅ **勾选框自动填充**：☐→☑ 根据案件条件自动判断
- ✅ **已用真实判决书验证**：(2024)冀0531刑初20号，10项赔偿金额全部精确匹配

## 快速开始

### 1. 模板文件

Skill已内置要素式起诉状模板（`templates/模板_民事起诉状.docx`），该模板依据《最高人民法院 司法部 中华全国律师协会关于印发部分案件起诉状答辩状示范文本的通知》（法〔2025〕82号）文件中的示范文本。如需使用其他法院模板，可通过 `--template` 参数指定。

### 2. 输入案件数据

```python
import sys
sys.path.insert(0, 'scripts')
from workflow_main import run_workflow

case_input = {
    "plaintiff_name": "张某某",
    "plaintiff_gender": "男",
    "plaintiff_age": 45,
    "case_type": "civil",           # civil=纯民事 | criminal_attached=刑事附带民事
    "injury_type": "disability",     # death=死亡 | disability=伤残 | property=仅财产损失
    "liability_type": "全责",        # 全责/主责/同责/次责/无责
    "standard_year": "2025",        # 适用赔偿标准年份
    "region": "urban",              # urban=城镇 | rural=农村
    "hospital_days": 45,
    "medical_fee": 52340.80,
    "nursing_days": 45,
    "lost_work_days": 180,
    "traffic_fee": 680,
    "disability_grade": 10,         # 1-10级
    "accident_detail": "2025年3月15日...",
    "defendants_person": [{"name": "李某某"}],
    "defendants_company": [{"name": "某某保险公司"}],
    "court": "河北省邢台市桥西区人民法院",
}

result = run_workflow(case_input, template_path="模板_民事起诉状.docx")
```

### 3. 命令行运行

```bash
python scripts/workflow_main.py --input case_data.json --template 模板_民事起诉状.docx
```

### 4. 三期评定查询

```python
from scripts.three_period_data import lookup_three_period, lookup_three_period_multi

# 单处损伤
result = lookup_three_period("左股骨颈骨折")
# → 误工180-270天(中值225) 护理60-120天(中值90) 营养60-90天(中值75)

# 多处损伤
results = lookup_three_period_multi("右胫腓骨粉碎性骨折伴踝关节脱位")
# → 匹配"胫腓骨骨折"+"踝关节脱位"，分别给出三期
```

## 案件类型支持

### 伤情类型
| 类型 | 影响的赔偿项目 |
|------|------------|
| 死亡 | 死亡赔偿金+丧葬费（无残疾赔偿金） |
| 伤残 | 残疾赔偿金+精神损害抚慰金+鉴定费 |
| 仅财产损失 | 仅车辆损失等 |

### 案件类型
| 类型 | 特殊规则 |
|------|---------|
| 纯民事 | 支持精神损害抚慰金 |
| 刑事附带民事 | ❌不支持精神损害抚慰金（刑诉法解释第192条） |

### 责任比例
| 类型 | 比例 |
|------|------|
| 全责 | 100% |
| 主责 | 70% |
| 同责 | 50% |
| 次责 | 30% |

## 注意事项

1. **日标准可自定义**：不同法院住院伙食可能50或100元/天，通过 `hospital_meal_daily` 参数覆盖
2. **勾选框原理**：☐在XML中是独立`<w:t>`元素，必须用XML段落遍历替换，不能用简单字符串替换
3. **精神损害抚慰金**：河北省标准十级1000-5000元...一级40000-50000元，死亡50000元
4. **非医保扣减**：河北省标准最高不超过10%
5. **模板必须使用法院原件**：不能从零生成，样式无法还原
